HTML & CSS Quick Reference Guide

HTML Tags

<h1> - <h6>: Headings

It is used to define headings, where <h1> is the largest and <h6> is the smallest.

<h1>A Heading</h1>

<p>: Paragraph

This defines a paragraph of text.

<p>A paragraph.</p>

<a>: Anchor (Link)

Creates a hyperlink to another webpage.

<a href="https://www.facebook.com">Visit Facebook</a>

<img>: Image

Enters an image.

<img src="image.jpg" alt="A description of the image">

<ul>, <ol>, <li>: Lists

Creates ordered and unordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table>: Table

Defines a table structure.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Abeera</td>
        <td>19</td>
    </tr>
</table>

<form>: Form

Creates an input form.

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</form>

<div> & <span>: Containers

<div> is a block-level container, whereas <span> is an inline container.

<div>This is a block element</div>
<span>This is an inline element</span>

<button>: Button

Creates a clickable button.

<button>Click on Me</button>

<blockquote>: Block Quote

Defines a section quoted from a different source.

<blockquote>A quote.</blockquote>

CSS Properties

color

It changes the text color.

p {
    color: #898989;
}

background-color

Defines the background color of an element.

body {
    background-color: #f4f4f4;
}

font-family

Specifies the font style.

body {
    font-family: 'Arial', sans-serif;
}

border

Defines a border around an element.

div {
    border: 4px solid #f4f4f4;
}

padding

Defines inner spacing within an element.

button {
    padding: 10px 20px;
}

margin

Defines outer spacing around an element.

h1 {
    margin-bottom: 20px;
}

display

Controls how elements are displayed.

span {
    display: inline-block;
}

box-shadow

Adds a shadow around an element.

div {
    box-shadow: 2px 2px 10px rgba(0,0,0,0.1);
}
